# QPST automatin

## 참고문서
80-PA177-1 RevA

## 개요
QPST API를 이용하여 QPST 기능을 custom 하여 자동화할 수 있다.

## 위치
C:\Program Files (x86)\Qualcomm\QPST\Automation Samples

## Key API
1. Add COM Port
2. Remove COM Port
3. Enable/Disable Port
4. Get port 
5. Get portlist
6. Get NV item.
7. restore qcn
8. Send Command
9. Get information
10. Phone reset
11. EFS file remove and write

## 추가 구현
- EFS 가져오기, 그리고 정보 읽기
- MIPI 정보, RFSW debug 정보
